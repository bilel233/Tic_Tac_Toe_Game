# Tic Tac Toe en Python

Bienvenue dans ce projet de Tic Tac Toe en Python ! Ce jeu classique est une excellente occasion d'explorer les concepts de programmation et de logique.

## Comment jouer

1. **Installation :** Assurez-vous d'avoir Python installé sur votre machine. Si ce n'est pas le cas, vous pouvez le télécharger sur [le site officiel de Python](https://www.python.org/).

2. **Clonage du projet :** Clonez ce dépôt sur votre machine en utilisant la commande suivante dans votre terminal.

    ```bash
    git clone https://github.com/tonnomdutilisateur/tic-tac-toe-python.git
    ```

3. **Exécution :** Accédez au répertoire du projet et exécutez le fichier principal.

    ```bash
    cd tic-tac-toe-python
    python main.py
    ```

4. **Jouez !** Suivez les instructions à l'écran pour jouer au Tic Tac Toe.

## Fonctionnalités

- [ ] Plateau de jeu 3x3.
- [ ] Deux joueurs : X et O.
- [ ] Alternance automatique entre les joueurs.
- [ ] Détection du gagnant ou d'une égalité.
- [ ] Affichage du plateau après chaque coup.
- [ ] Possibilité de rejouer.
